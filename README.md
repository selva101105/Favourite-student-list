# Favourite Student List

A simple React project to manage students and their favourites.  
This project allows you to:

- View all students
- Add students to favourites
- Remove students from favourites
- See which students are already added

Built using **React**, **React Router**, **Tailwind CSS**, and **Context API**.

---

## Features

1. **Student List**:  
   - Displays all students with details: ID, Name, Department, Year  
   - Each student has an **Add to Favourite** button  
   - Button disables automatically if student is already in favourites  

2. **Favourite Students**:  
   - Displays all favourite students  
   - Each student has a **Remove** button  

3. **Context API**:  
   - Used to manage student and favourite data globally  

4. **Tailwind CSS**:  
   - Used for responsive and clean UI  

---

## Installation

1. Clone the repository:
git clone <https://github.com/selva101105/Favourite-student-list.git>

2. Install dependencies:

3.Run the project:
npm run dev
